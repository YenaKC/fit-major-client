# FIT MAJOR

Premium gymwear e-commerce platform built with the MERN Stack.

## Features

- Authentication (Signup / Login)
- Product Catalog
- Search & Category Filter
- Shopping Cart
- Order Creation
- Responsive Design
- JWT Authentication
- MongoDB Atlas Database
- Stripe Checkout Integration

## Tech Stack

Frontend
- React
- React Router
- Axios
- CSS

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt
- Stripe

## Installation

Frontend

npm install
npm run dev

Backend

npm install
npm start

## Environment Variables

Frontend

VITE_API_URL=

Backend

MONGO_URI=
JWT_SECRET=
STRIPE_SECRET_KEY=
